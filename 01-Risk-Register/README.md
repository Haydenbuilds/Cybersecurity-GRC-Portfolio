# Project 01: Cybersecurity Risk Register - ApexDigital Fintech

## Project Overview
This project involves a comprehensive risk assessment for **ApexDigital**, a fictional Fintech startup. The goal is to identify, analyze, and prioritize cybersecurity risks to ensure data protection and regulatory compliance.

## Methodology
- **Framework:** Aligned with **NIST SP 800-30** standards.
- **Risk Scoring:** Risk Score = Likelihood (1-5) x Impact (1-5).
- **Focus Areas:** Ransomware, Data Breaches, Phishing, and Regulatory Compliance (e.g., GDPR/Decree 13).

## Key Highlights
- Identified **10 critical risks** across technical and operational domains.
- Developed specific **Mitigation Plans** utilizing tools like SIEM (Splunk) and IPS (Suricata).
- Prioritized resources based on a **Risk Heatmap** model.

## Deliverables
- `ApexDigital_Risk_Register.xlsx`: Full detailed risk register with scoring formulas and color-coded categorization.
