# Project 02: Vendor Security Assessment - Anthropic

## Executive Summary
This assessment evaluates **Anthropic** as a potential AI service provider. Due to industry standards, the full SOC 2 Type II report is restricted, so this review is based on their **Public Trust Center artifacts (SOC 3, ISO 27001, FedRAMP High, and HIPAA Compliance).**

## Compliance Posture
- **SOC 2 Type II & SOC 3:** Certified for Security, Availability, and Confidentiality.
- **ISO 27001:2022:** Globally recognized standard for Information Security Management Systems (ISMS).
- **FedRAMP High:** Meets the highest security requirements for US Government data.
- **HIPAA:** Authorized for processing Protected Health Information (PHI).

## Key Security Controls
- **Data Encryption:** TLS 1.2+ for data in transit; AES-256 for data at rest.
- **Subprocessor Oversight:** Uses AWS/GCP/Azure (all Tier-1 providers with their own SOC 2 reports).
- **Vulnerability Management:** Annual third-party penetration tests and bug bounty program.
